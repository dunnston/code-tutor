## Notes & Conventions

### Naming Conventions
- **Python**: Use fantasy/RPG themes (Scroll, Quest, Forge, etc.)
- **GDScript**: Use game dev themes (Node, Scene, Signal, etc.)
- **JavaScript**: Use web dev themes (Console, DOM, Event, etc.)
- **C#**: Use OOP/enterprise themes (Class, Method, Property, etc.)

### Difficulty Progression
- Lessons 1-10: Difficulty 1-3 (Very Basic)
- Lessons 11-20: Difficulty 3-5 (Intermediate)
- Lessons 21-30: Difficulty 5-7 (Advanced Fundamentals)

### XP Rewards
- Simple lessons (1-10): 50-100 XP
- Medium lessons (11-20): 100-150 XP
- Complex lessons (21-25): 150-200 XP
- Project lessons (26-30): 200-500 XP

### Validation Tests
Use a mix of:
- `output_contains` - Check console output
- `code_contains` - Check for specific code patterns
- `function_exists` - Verify function is defined
- `variable_exists` - Verify variable is defined
- `function_returns` - Check function return value
- `output_line_count` - Verify correct number of lines
